折叠面板
可以同时展开多个面板，这里默认展开第2个 
<template>
  <div class="collapse-demos">
    <c-collapse v-model="activeKey">
      <c-collapse-panel panelKey="1" header="Panel header1">
        {{ content }}
      </c-collapse-panel>
      <c-collapse-panel panelKey="2" header="Panel header2">
        {{ content }}
      </c-collapse-panel>
      <c-collapse-panel panelKey="3" header="Panel header3">
        {{ content }}
      </c-collapse-panel>
    </c-collapse>
  </div>
</template>

<script>
export default {
  data() {
    return {
      activeKey: ['2'],
      content:
        'Performs a deep comparison between two values to determine if they are equivalent.',
    };
  },
  watch: {
    activeKey(newV) {
      console.log(newV);
    },
  },
};
</script>

# API

# Collapse

参数 说明 类型 默认值
activeKey(v-model) 当前活跃的 tab 面板的 key string或string[] 默认无，手风琴模式下默认第一个
bordered 带边框风格的折叠面板 boolean -
accordion 手风琴模式 boolean false

# 事件

事件名称 说明 回调参数
change 切换面板时的回调 function (key)

# Collapse.Panel

参数 说明 类型 默认值
header 面板头内容 string 或slot -
showArrow 是否展示面板上的箭头 boolean true
disabled 面板是否禁用 boolean false
panelKey 对应activeKey string -
